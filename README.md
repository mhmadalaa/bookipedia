## Installation

- To install bookipedia-api, follow these steps:

   ```bash
   $ git clone <repo-link>
   $ cd <app-directory>
   $ npm install
   ```

 - Set up environment variables in .env file

- Start the application:
   ```bash
   npm run start:dev
