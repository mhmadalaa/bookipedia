## Installation ðŸš€

To install Bookipedia-API, follow these simple steps:

1. Clone the repository:
   ``` bash
   $ git clone <repo-link>
   $ cd <app-directory>
   ```

2. Install dependencies:

   ``` bash
   $ npm install
   ```
3. Set up environment variables in the `.env` file.

4. Start the application in development mode:
   ```bash
   $ npm run start:dev
   ```
